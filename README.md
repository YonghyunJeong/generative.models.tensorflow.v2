This repository is a collection of various generative models (GAN, VAE, Flow based, etc)

## Getting Started

### Prerequisites
* [`TensorFlow`](https://www.tensorflow.org)
* Python 3.6
* Python libraries:

* Jupyter notebook
* OS X and Linux (Not validated on Windows OS)
